# Руководство Администратора (Boterator)

## Установка и запуск
1. Склонируйте репозиторий.
2. Выполните скрипт `sudo bash install.sh`. Следуйте инструкциям на экране.
3. Убедитесь, что служба запущена: `sudo systemctl status boterator`.

## Управление
Вся работа с пользователями и тарифами происходит через **CRM Администратора** в Telegram (TWA). Откройте бота и нажмите "CRM Администратора".

- **Дашборд:** Показывает текущую статистику дохода и оттока пользователей.
- **Тарифы:** Позволяет создавать новые тарифы (в т.ч. триальные или скрытые).
- **Пользователи:** Управление ручной выдачей подписок.
- **Рассылки:** Отправка сообщений всем пользователям или сегментам по тарифам.

## Обновление
Для обновления бота выполните `sudo bash update.sh`. Все настройки в папке `DEVELOPE` будут сохранены.
